# 📊 Superstore Sales Analysis (Tableau)

## 📌 Project Overview
This project presents a full end-to-end data analysis using the Superstore dataset.

The workflow includes:
- Data exploration
- Data cleaning
- Data modeling
- Data visualization using Tableau

The goal is to extract meaningful business insights related to sales, profit, customers, and regional performance.

## 🔍 Analysis Process

### 1. Data Exploration
- Identified missing values
- Checked data types
- Explored key metrics (Sales, Profit, Quantity)

### 2. Data Cleaning
- Handled null values
- Fixed inconsistencies
- Created calculated fields

### 3. Data Modeling
- Built relationships between tables
- Optimized data structure for analysis

### 4. Data Visualization (Tableau)
Created interactive dashboard including:
- KPI cards (Sales, Profit, Orders)
- Sales trends over time
- Regional performance
- Top products & categories
- Customer segmentation

---

## 📊 Key Insights
- High sales do not always mean high profit
- Certain regions generate more profit than others
- A small number of products drive most revenue
- Discounts significantly impact profitability
